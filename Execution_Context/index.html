Execution Context

What is execution Context?

Java Script people love to keep fancy names...The environment say it function, scope, variable etc etc all together under one environment (Matlab seedha hai boss environment jiske andar jo kuch bhi sab chal sake..)

Execution Context has 2 components (Execution ke do hisse hai)

  1. Memory Component
  2. Execution Component

Memory Component

Memeory is being allocated during this phase.. mtlb kya?
For example :- When you declare a variable
var x = 14;
You declared a variable and some memory is allocated to it

Code Component / Execution Component

Code is being executed during this phase.. mtlb kya?
Example :-
var x = 14;
var y = x;
console.log(y);
Code is being executed

*Java Script is synchronous Multithreaded language* (Matlb line by line harr thread ko execute karta hai)

*Hoisting :- just that you are able to access variable and functions before intializing the code..*
